# 〰️ Squiggle School

**Printable worksheets that kids actually enjoy.**

Open it. Click once. Print. Done.

---

## What is it?

Squiggle School is a single HTML file that generates a different printable worksheet every time you click. No app to install, no account to create, no internet required after the first load.

It was built for our homeschooling family and shared here in case it's useful for yours.

---

## How to use it

1. **[Open Squiggle School →](https://fromchaosimportcalm.github.io/squiggle-school)**
2. Pick an activity from the sidebar (or hit **✨ Surprise Me**)
3. Click **🎲 New Sheet** to generate a fresh worksheet
4. Click **🖨 Print** or `Ctrl+P` / `Cmd+P`
5. Hand it to a child with a pencil

That's it.

## Features

- **Every worksheet is different** — click New Sheet to regenerate
- **Difficulty levels** — Easy, Medium, Tricky (affects mazes, numbers, word length)
- **Surprise Me** — randomly picks an activity with a bias toward creative ones
- **Print-ready** — sidebar and toolbar disappear when you print, leaving a clean A4 page
- **Works offline** — save the file locally and it runs without internet (fonts fall back gracefully)
- **No tracking, no ads, no accounts**

---

## Offline use

Right-click the page → **Save As** → save the `.html` file anywhere on your computer. Open it any time — it works completely offline.

---

## For developers

The entire app is a single `.html` file with no build step, no dependencies, and no framework. All worksheet generators are vanilla JS producing SVG and HTML strings. If you want to add an activity or tweak something, open the file in any text editor and find the relevant `gen...()` function.

Pull requests welcome.
